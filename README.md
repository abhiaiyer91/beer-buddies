# beer-buddies

Prototype app to show off `GraphQL Subscriptions`.

## Getting Started

1. Run Redis w/ docker

2. Run the `eventador`

`cd packages/eventador`
`npm run start`

3. Run the `product-server`

`cd packages/product-server`
`npm run start`

4. Run the `product-client`

`cd packages/product-client`
`npm run start`
